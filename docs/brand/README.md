# Mumega and Mupot brand assets

This directory contains the approved identity concepts and the production Mupot
icon package.

## Approved marks

- `mupot-logo-planter-mu-v1.png`: Mupot's ochre planter, Greek `mu`, and two
  movement lines.
- `mumega-logo-omega-flipped-v2.png`: Mumega's Greek `mu` nested inside an
  omega, using the shared ochre and editorial wordmark.

Mupot is the governed vessel/product. Mumega is the broader parent identity.
The shared `mu`, palette, and typography make them a family without repeating
the same silhouette.

## Production Mupot icons

`production/` contains browser, Apple, Android/PWA, maskable, and Microsoft
sizes derived pixel-for-pixel from the approved Mupot artwork. See
`production/README.md` for the asset matrix and integration markup.

Regenerate the icon package with:

```sh
npm run brand:export
```
